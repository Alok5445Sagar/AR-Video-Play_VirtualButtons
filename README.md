# 📖 Overview

The **AR Vid Play Virtual Buttons** project is an interactive Augmented Reality application developed using **Unity 3D** and **Vuforia Engine**. The application combines **marker-based AR tracking** with **virtual button interactions** to control multimedia playback dynamically in an immersive AR environment.

Users can scan image targets and interact with virtual button regions to:
- ▶️ Play videos
- ⏸️ Pause multimedia
- 🔄 Trigger AR interactions
- 🎮 Control digital content in real time

This project demonstrates:
- Augmented Reality interaction systems
- Virtual button implementation
- Real-time video rendering
- Marker-based tracking
- Interactive multimedia experiences

# ⚙️ System Workflow

```text
Camera Detects Image Target
              ↓
Vuforia Initializes Tracking
              ↓
Virtual Buttons Activated
              ↓
User Interacts with Buttons
              ↓
Video Playback Triggered
              ↓
Real-Time AR Rendering
```

---

# 📸 How It Works

1. The user scans the predefined image target using the mobile camera.
2. Vuforia detects and tracks the marker in real time.
3. Virtual buttons are placed on specific regions of the target.
4. User interaction with the buttons triggers video playback actions.
5. The video is displayed dynamically within the AR environment.
